# 📊 Wikipedia Analysis and Classification

**Client:** Wikidata Insights (for Wikimedia) | **Year:** 2024

## 📝 Project Overview
This project was developed to analyze the vast ecosystem of Wikipedia articles and build a **Machine Learning** system capable of automatically classifying new content. The goal is to optimize content categorization and extract strategic linguistic insights.

## 🎯 Main Objectives
* **Exploratory Data Analysis (EDA):** Understanding category distribution (e.g., *Science, Politics, Technology*) and extracting linguistic features such as article length and word frequencies.
* **Automatic Classification (NLP):** Developing and training a Natural Language Processing model to predict and assign the correct thematic category to newly written articles.
* **Strategic Insights:** Identifying language trends and editorial opportunities across different topics.

## 🚀 Future Applications & Impact
* **Editorial Automation:** Seamless integration into Wikimedia's CMS to auto-suggest categories during the drafting of new articles.
* **Quality Assurance:** Detecting and fixing historically miscategorized articles.
* **Trend Analysis:** Monitoring how language and terminology evolve within specific categories over time.
